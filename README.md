# Aalekh

**Architecture visualization and linting for Android and Kotlin Multiplatform projects.**

Aalekh is a Gradle plugin that helps teams **see, guard, and understand their architecture** as projects grow. It extracts module dependencies from Android and Kotlin Multiplatform builds and generates interactive visualizations that reveal how your codebase is structured.

As multi-module projects evolve, architectural boundaries can slowly erode - feature modules start depending on each other, clean architecture layers leak, and circular dependencies appear. Aalekh makes these issues visible and, over time, enforceable through architecture rules and automated checks.

The goal of Aalekh is simple: **help teams keep their architecture intentional, observable, and maintainable.**
